Commands to create WebAPI image
---------------------------------------
1) docker build -t mssqlwebapi -f Dockerfile .
2) docker tag mssqlwebapi ateet1989/mssqlwebapi:v3
3) docker images
4) docker login
5) docker push ateet1989/mssqlwebapi:v3
